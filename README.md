# Electiva DISEÑO DE INTERFACES - 2025-1 - Santiago Montoya Baiter


## Estructura del Proyecto
- `index.html`: Archivo principal que carga todos los componentes
- `css/main.css`: Estilos principales del sitio
- `components/`: Directorio que contiene los componentes HTML separados:
  - `header.html`: Encabezado del sitio
  - `hero.html`: Sección principal de presentación
  - `skills.html`: Sección de habilidades
  - `stats.html`: Estadísticas
  - `try-lesson.html`: Sección para probar lecciones
  - `community.html`: Información sobre comunidad
  - `more-features.html`: Características adicionales
  - `testimonials.html`: Testimonios
  - `projects.html`: Proyectos realizados
  - `pricing.html`: Precios de servicios
  - `financing.html`: Opciones de financiamiento
  - `footer.html`: Pie de página

## Instalación
1. Clona este repositorio
2. Abre el archivo `index.html` en tu navegador con un servidor como Live Server

## Desarrollo
Para modificar el sitio:
1. Edita los archivos individuales en la carpeta `components/`
2. Modifica los estilos en `css/main.css`
3. Actualiza el contenido según sea necesario
